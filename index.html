Inicio Vetserver

Vetserver

Tus datos en las mejores manos.

En VetServer sabemos lo complicado que puede ser el manejo de información de un centro veterinario en crecimiento, tener el almacenamiento de información de clientes, mascotas e historial clínico de forma adecuada puede ser complejo y tedioso de manejar. Por esto nuestro proyecto Vetserver está enfocado en el desarrollo de un aplicativo web que apoye el Centro Veterinario Villa Paula, el cual sea accesible, práctico y visualmente atractivo; el aplicativo web cuenta con una página principal que sea accesible para todos los clientes donde puedan encontrar toda la información básica de la veterinaria y cuenta con un apartado solo para el ingreso del personal del centro veterinario donde podrán almacenar toda la información que requieren y será establecida completamente en formato digital.

Especificaciones del aplicativo

Las tecnologías que fueron usadas para el desarrollo:

  • Lenguaje de programación: Java script
  • Lenguaje de marcado: HTML
  • Lenguaje de hojas de estilo en cascada: CSS
  • Base de datos: Mysql
  • Conexión servdior: Node Js
  • Repositorio en la nube: Heroku
  • Sistema de control de versiones: GitHub
